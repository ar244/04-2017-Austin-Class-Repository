Run your websites on a public server in 2 minutes

Ngrok is a secure way to make your website accessible on the web.

First, install it with:

npm install -g ngrok

Run the server you want to expose on your local machine. Let's say you have a nodejs server named server.js:

node server.js

You can run ngrok like:

ngrok http -host-header=localhost {port}

Let's say the node server is now running on port 8080. You can run it publicly with:

ngrok http -host-header=localhost 8080

Notice the output has a line like:

Forwarding     http://ad5444a3.ngrok.io -> localhost:8080

Visit that http://{random_id}.ngrok.io address that your terminal outputted and you should now see your website. Test it on your phone

Some tutorials on ngrok won't give the full command, but make sure to give the host-header flag.
